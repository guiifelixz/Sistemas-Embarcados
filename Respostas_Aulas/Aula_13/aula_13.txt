1. Apresente 5 sistemas operacionais suportados pelo Raspberry Pi, e algumas de suas características.
	Arch Linux: distribuição linux bem simples e otimizada, ideal para caráter mais técnico.
	Raspbian: distribuição padrão do raspberry, costuma ser mais utilizada e tem algumas aplicações padrões para tal.
	Windows 10: distribuição desenvolvida pela Microsoft voltado para IOT.
	Ubuntu: Versão muito próxima do Ubuntu para desktops. Ela possui uma interface gráfica próxima da tradicional e recursos interessantes para um usuário com conhecimento menos técnico.
	OSMC: sistema operacional voltada para utilização de mídias (vídeo, música, etc).

2. Apresente as formas de instalação de sistemas operacionais para o Raspberry Pi.
	Via cartão SD bootável.

3. Apresente 3 formas de desenvolvimento de software para o Raspberry Pi.
	Através da compilação de acesso remoto, programando atraés de outro computador; Através de teclado e mouse físicos no próprio raspberry; Gerando o executável compilado, via crossed-compilation para o raspberry.

4. Apresente 3 formas de acesso remoto ao Raspberry Pi.
	Via protocolo VNC, RTD (protocolo da Microsoft) e SSH.

5. Apresente as formas possíveis de compilação de código em C para o Raspberry Pi.
	Via terminal, Visual Studio e Codeblocks.

6. Apresente as formas possíveis de compilação de código em Python para o Raspberry Pi.
	Via terminal, IDLE3.
